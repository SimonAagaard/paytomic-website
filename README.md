# Paytomic Website

This repository contains the source code for [https://paytomic.dk](https://paytomic.dk), the official website for Paytomic.

## Project Structure

- **frontend/** - Angular 21 application with SSR (Server-Side Rendering)
- **backend/** - .NET API for contact form and backend services

## Frontend

Built with Angular 21, featuring:
- Server-Side Rendering (SSR) for improved SEO and performance
- Standalone components architecture
- NG-Zorro Ant Design components
- Tailwind CSS for styling
- Responsive design with mobile-friendly navigation

## Backend

ASP.NET Core API running on .NET 10:
- JWT authentication
- Email service for contact forms
- CORS configuration for production domains
